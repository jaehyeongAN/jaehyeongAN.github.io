3. 단위 계단 함수

  • u(-1) = 0, u(1) = 1, u(0) = 1
  • 단위 계단 함수는 원점에서 불연속 즉, ‘미분 불가능’
  • 미분 불가능한 특성으로 인해 신경망의 활성화 함수로 잘 사용되지 않음.
